\# Comandos Básicos do Git



\## 📌 Objetivo



Documentar os principais comandos utilizados durante o estudo dos fundamentos do Git.



\---



\## 🛠️ Inicialização de Repositório



\### git init

Inicia um novo repositório Git no diretório atual.



Utilizado para versionar projetos localmente.



\---



\### git clone \[URL]

Clona um repositório remoto existente para o ambiente local.



Permite sincronização entre o repositório local e remoto.



\---



\## 📦 Controle de Alterações



\### git add .

Adiciona arquivos modificados à área de staging.



\---



\### git commit -m "mensagem"

Registra alterações no histórico do repositório.



\---



\### git status

Exibe o estado atual do diretório de trabalho e arquivos rastreados.



\---



\### git diff

Mostra diferenças entre arquivos modificados.



\---



\## 🌐 Repositórios Remotos



\### git remote -v

Lista os repositórios remotos configurados.



\---



\### git pull

Atualiza o repositório local com alterações remotas.



\---



\### git push

Envia commits locais para o repositório remoto.
